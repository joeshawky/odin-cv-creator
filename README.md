# 📝 CV Creator (React)

A browser-based CV (resume) builder developed during the React module of **The Odin Project**.

## ✨ Project Overview

This project was built using React to manage dynamic form state and real-time updates. It allows users to input their personal details, education history, and work experience, and preview a structured CV layout as they type.

Users can then print their completed CV as a PDF using their browser’s print functionality.

## 🧠 Core Concepts Demonstrated

- Component-based architecture in React  
- State management with `useState`  
- Form input handling and two-way data binding  
- Conditional rendering and dynamic content updates  
- Print-friendly styling with CSS

## 📐 Features

- Real-time CV preview as you type  
- Editable sections for personal info, education, and work experience  
- Print to PDF support using browser print dialog  
- Clean and responsive design with CSS  
- Organized, readable code with clear component separation
